# templates/

> Genbrugelige faglige og projektstyringsmæssige skabeloner.

## Formål

Sikre konsistente leverancer på tværs af projekter og agenter.

## Undermapper

| Mappe | Indhold |
|-------|---------|
| `load-lists/` | Load list-skabeloner |
| `work-descriptions/` | Arbejdsbeskrivelser |
| `tender-lists/` | Tilbuds- og mængdelister |
| `qa/` | QA-tjeklister og review-skabeloner |
| `diagrams/` | Diagramskabeloner |
| `calculations/` | Beregningsskabeloner |
| `decision-notes/` | Beslutningsnotater |
| `risk-logs/` | Risikolog-skabeloner |

## Hvad der ikke må gemmes her

- Projekt-specifikke udgaver
- Færdige leverancer
- Personfølsomme oplysninger

# Command: Start projektdagbog

> Opret eller opdater dagens tekniske dagbog
> Trigger: "Start dagbog" / "Opdater dagbog"

## Workflow

1. Åbn `brain/diary/YYYY-MM-DD.md` (opret hvis den ikke findes).
2. Notér:
   - Dato og projekt
- Dagens fokus
- Hvad der er blevet gjort
- Åbne punkter
- Næste skridt
3. Link til relevante `brain/tasks/`-poster.
4. Gem filen.

## Output

- Opdateret dagbogsfil med status og næste skridt.

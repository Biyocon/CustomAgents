# Command: Lav beslutningsnotat

> Udarbejd beslutningsklart notat
> Trigger: "Lav beslutningsnotat"

## Workflow

1. Beskriv baggrund og problemstilling.
2. Angiv alternativer.
3. Vurder fordele/ulemper ved hvert alternativ.
4. Anbefal én løsning med begrundelse.
5. List åbne punkter og næste handlinger.

## Output

- Beslutningsnotat med anbefaling, risici og næste skridt.

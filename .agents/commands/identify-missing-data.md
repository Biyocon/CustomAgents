# Command: Hvad mangler vi?

> Identificér manglende data og afklaringer
> Trigger: "Hvad mangler vi?" / "Manglende data"

## Workflow

1. Læs opgavebeskrivelse og tilgængeligt materiale.
2. Identificér alle antagelser.
3. For hver antagelse: er der behov for bekræftelse?
4. List konkrete manglende data med ejer og deadline.
5. Forslag til hvordan data indhentes.

## Output

- Liste over manglende data
- Ansvarlige/akter
- Prioritet
- Forslag til indhentning

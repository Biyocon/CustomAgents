# Command: Kør Ralph-loop

> Iterativt arbejde med plan, udførelse, kontrol, forbedring
> Trigger: "Kør Ralph-loop" / "Ralph-loop"

Ralph-loopet er en PDCA-lignende cyklus tilagent-arbejde:

1. **Plan** — forstå opgaven, identificér risici, vælg agenter/skills
2. **Do** — udfør opgaven trin for trin
3. **Check** — review resultatet mod succeskriterier
4. **Act** — forbedr, dokumentér læring, opdater Brain

## Workflow

1. Læs opgaven og Brain-kontekst.
2. Vælg relevant agentprofil og skills.
3. Udfør opgaven.
4. Review output med `shared-quality` eller `qa`.
5. Opdater `brain/personal/lessons-learned.md` og `brain/tasks/completed.md`.
6. Rapportér status og åbne punkter.

# Command: Lav tilbudsliste

> Udarbejd tilbudsliste med poststruktur
> Trigger: "Lav tilbudsliste"

## Workflow

1. Indsaml mængder og beskrivelser.
2. Strukturér poster med enhed, mængde, beskrivelse.
3. Marker usikre mængder og antagelser.
4. Referér til `templates/tender-lists/`.

## Outputformat

| Post | Beskrivelse | Enhed | Mængde | Antagelser | Kommentar |
|------|-------------|-------|--------|------------|-----------|

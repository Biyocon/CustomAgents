# Command: Lav risikolog

> Identificér og strukturér risici
> Trigger: "Lav risikolog" / "Risikolog"

## Workflow

1. Identificér risici baseret på opgave og input.
2. Vurder sandsynlighed og konsekvens.
3. Angiv ejer og mitigerende handlinger.
4. Marker kritiske risici.

## Outputformat

| ID | Risiko | Kategori | Sandsynlighed | Konsekvens | Ejer | Mitigering | Status |
|----|--------|----------|---------------|------------|------|------------|--------|

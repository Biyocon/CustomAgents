# Command: Lav kravmatrix

> Generér struktureret kravmatrix fra input
> Trigger: "Lav kravmatrix" / "Kravmatrix"

## Workflow

1. Indsaml krav fra input (dokument, regelsæt, udbud).
2. Kategorisér krav efter type: funktionelle, tekniske, juridiske, sikkerhed, kvalitet.
3. Tildel unikke krav-ID'er.
4. Angiv kilde, ansvarlig, status, afhængigheder.
5. Marker åbne punkter.

## Outputformat

| ID | Krav | Kategori | Kilde | Ansvarlig | Status | Kommentar |
|----|------|----------|-------|-----------|--------|-----------|

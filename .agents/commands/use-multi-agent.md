# Command: Brug multi-agent

> Aktivér relevante specialistroller og syntetisér resultatet
> Trigger: "Brug multi-agent" / "Kør multi-agent"

## Workflow

1. **Forstå opgaven**
   - Hvad skal løses?
   - Hvilke fagdiscipliner berøres?

2. **Vælg specialistagenter**
   - Vælg 2-5 agenter fra `.agents/agents/` eller `.vscode/.codex/agents/`
   - Prioritér Banedanmark-roller ved BaneByg-opgaver

3. **Fordel delopgaver**
   - Giv hver agent en konkret delopgave
   - Angiv input og forventet output

4. **Saml og syntetisér**
   - Kombinér input fra alle agenter
   - Løs konflikter
   - Formuler én samlet konklusion

5. **Dokumentér**
   - Hvilke agenter deltog
   - Hvad blev de enige om
   - Åbne punkter

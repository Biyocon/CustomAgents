# Command: Lav arbejdsbeskrivelse

> Udarbejd teknisk arbejdsbeskrivelse
> Trigger: "Lav arbejdsbeskrivelse"

## Workflow

1. Afklar omfang, grænser og forudsætninger.
2. Beskriv materialer, metoder og kvalitetskrav.
3. Referér til standarder og specifikationer.
4. Marker antagelser og åbne punkter.
5. Strukturér efter `templates/work-descriptions/`.

## Output

- Klar tekst til dokument eller udbud.

# brain/personal/

> Personlig brugerhukommelse til Custom agent-harnesset.

## Formål

Opbygge og vedligeholde en profil af brugeren, så agenter kan tilpasse sig præferencer, rolle og læring.

## Indhold

| Fil | Formål |
|-----|--------|
| `user-profile.md` | Brugerens rolle, baggrund, ansvarsområder |
| `preferences.md` | Outputformater, sprog, tone, værktøjer |
| `lessons-learned.md` | Læring fra tidligere opgaver |
| `open-decisions.md` | Åbne beslutninger der påvirker agent-arbejde |

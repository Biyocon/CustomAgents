# Domaenesprog og forkortelser

## Banedanmark / Jernbanesikkerhed

| Forkortelse | Betydning | Kontekst |
|-------------|-----------|----------|
| **BBTR** | BaneByg Teknisk Raadgivning | Raadgivningsenhed under Banedanmark |
| **BBE** | BaneByg Enterprise | Platform/IT-system til projektstyring |
| **BKP** | Brutto Kontrolplan | Kontrolplan for jernbaneprojekter |
| **ABR18** | Almindelige Betingelser for Raadgivningsydelser 2018 | Standardkontrakt for raadgivere |
| **AB18** | Almindelige Betingelser for bygge- og anlaegsarbejder 2018 | Standardkontrakt for entreprenoerer |
| **SR** | Signalreglement | Trafikalt regelsaet for signalsystemer |
| **ORS** | Overenskomst om Rejsning af Spor | Trafikalt regelsaet for sporarbejde |
| **ORF** | Overenskomst om Rammer for Foering af Tog | Trafikalt regelsaet for togfoering |
| **CSM-RA** | Common Safety Method for Risk Evaluation and Assessment | EU-metode for risikovurdering i jernbanesikkerhed |
| **TSI** | Technical Specifications for Interoperability | Tekniske specifikationer for interoperabilitet |
| **SSB** | Supplerende SikkerhedsBestemmelser | Lokale tillaeq til SR/ORS/ORF |
| **SRAC** | Safety Risk Acceptance Criteria | Risikoacceptkriterier for sikkerhed |
| **TC** | Trafikcirkulaere | Cirkulaere fra Banedanmark om trafikale forhold |
| **TKP** | Total Kontrol Plan | Overordnet kontrolplan |
| **KP** | Kontrol Plan | Delplan under TKP |
| **ATR** | ArbejdsTidsRegistrering | Tidsregistrering i projekter |
| **IPM** | Integrated Project Model | Integreret projektmodel |
| **DoD** | Definition of Done | Afleveringskriterier |
| **BDK** | Banedanmark | Organisationen selv (ogsaa brugt som praefiks) |

## Agent-harness / Tekniske begreber

| Begreb | Betydning |
|--------|-----------|
| **Harness** | Den samlede infrastruktur af agenter, skills, prompts og konfigurationer |
| **Agent** | En specialiseret AI-rolle med eget prompt, trigger-betingelser og arbejdsomraade |
| **Skill** | En genanvendelig evne — typisk et prompt eller en guide — der kan aktiveres af agenter |
| **Brain** | Projektets persistente kontekst- og hukommelsesarkiv |
| **Runbook** | En trin-for-trin driftsguide |
| **ADR** | Architecture Decision Record — dokumenteret arkitekturbeslutning |
| **MCP** | Model Context Protocol — protokol til vaerktoejsintegration |
| **Runtime** | Den aktive koerselskontekst (stier, prompts, skills) for agenter |
| **Kombi** | Referencekatalog med upstream-materiale og inspiration |
| **Promovering** | Kopiering af projekt-harness til global brugerprofil |

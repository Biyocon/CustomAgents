# plugins/

> Integrationer og adapters til eksterne værktøjer og systemer.

## Formål

Beskrive hvordan agenter interagerer med de værktøjer, som brugeren arbejder med dagligt.

## Indhold

| Fil | Værktøj |
|-----|---------|
| `sharepoint.md` | SharePoint |
| `projectwise.md` | ProjectWise |
| `teams.md` | Microsoft Teams |
| `outlook.md` | Outlook |
| `autocad.md` | AutoCAD |
| `microstation.md` | MicroStation |
| `febdoc.md` | FEBDOC |
| `simaris.md` | Simaris |

## Hvad der ikke må gemmes her

- Loginoplysninger eller API-nøgler
- Projekt-specifikke datafiler
- Personfølsomme oplysninger

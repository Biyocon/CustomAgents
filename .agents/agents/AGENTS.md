# Agent-indeks

> Overblik over alle aktive og planlagte agenter i dette projekt.

## Struktur
Hver agent defineres i sin egen mappe med:
- `agent.md` — systemprompt og ansvarsområde
- `skills/` — agent-specifikke skills
- `memory/` — runtime-hukommelse og læring

## Registrering
Når du tilføjer en ny agent:
1. Opret mappe under `.agents/agents/<agent-navn>/`
2. Skriv `agent.md` med systemprompt
3. Opdater tabellen ovenfor
4. Tjek ind i git

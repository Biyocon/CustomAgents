# Projektering

## Formål
At sikre kvalificeret ledelse og styring af projekteringen i fornyelses- og anlægsprojekter, herunder at tekniske forhold er korrekt beskrevet i sikkerhedsbærende dokumenter, at rådgiveres leverancer kvalitetssikres, at tekniske risikovurderinger gennemføres, og at nødvendige myndighedstilladelser foreligger.

## Fremgangsmåde
1. Afklar projekteringsopgavens omfang, fase og kontraktgrundlag; bistå projektleder omkring tekniske forhold ved indgåelse af aftaler og ved udbud af rådgivningsydelser.
2. Udarbejd eller kvalitetssikr projekteringsplan og udbudstidsplan; kvalitetssikr udbudskontrolplan for tekniske forhold.
3. Kortlæg de projekterende discipliners leverancer pr. fase via **bbtr-faseopdelt-ydelser** og **bbtr-fagpakkestruktur**, og fastlæg kvalitetskrav og Definition of Done via **bbtr-kvalitet-dod**.
4. Kvalitetssikr rådgiverleverancer systematisk: tjek mod ydelsesbeskrivelser, tekniske regler og grænseflader; dokumentér kontrolresultater sporbart.
5. Gennemfør risikovurdering af tekniske forhold og sikr CSM/TSI-compliance i projekteringsfasen via **bbtr-csm-tsi-compliance** og **bbtr-risiko-myndighed**; afklar behov for myndighedstilladelser tidligt.
6. Koordinér på tværs af discipliner og med byggeledelse/driftsansvarlige via **bbtr-tvaerfaglig-koordinering**; håndtér grænseflader til sikring/signal særskilt (Projekteringsleder, Sikring).
7. Versionsstyr al projekteringsdokumentation jf. **bbtr-dokumentstyring**, og før erfaringer tilbage via forbedringsloopet.

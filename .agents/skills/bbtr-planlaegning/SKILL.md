# Planlægning

## Formål
At sikre sammenhængende tids-, ressource- og afhængighedsplanlægning på tværs af BaneByg-projekter, så leverancer, fagpakker og tredjepartsprojekter koordineres, og kritiske veje og milepæle overvåges aktivt.

## Fremgangsmåde
1. Afklar planlægningsomfanget: hvilke projekter, fagpakker og tredjepartsgrænseflader indgår, og hvilke rammer (sporspærringer, driftshensyn, myndighedsfrister) planen skal respektere.
2. Nedbryd projektet i aktiviteter med varigheder, ansvarlige og afhængigheder; identificér kritisk vej og buffere.
3. Kortlæg ressourcebehov og -konflikter på tværs af fagpakker; brug prioriteringsmøder til at løse konflikter (se **bbtr-tvaerfaglig-koordinering**).
4. Indarbejd faseopdelte leverancer og deres deadlines fra **bbtr-faseopdelt-ydelser** i den samlede tidsplan.
5. Etabler milestone-tracking og fast fremdriftsrapportering; markér afvigelser fra baseline eksplicit og eskalér via projektstyringen (**bbtr-projektstyring**).
6. Opdater planen versionsstyret ved ændringer jf. **bbtr-dokumentstyring**, og kommunikér ændringer via mødestrukturen.
7. Ved planlægning med trafikal berøring (køreplaner, ROMAN, S-bane-infrastruktur) inddrag de relevante trafikale planlægningsroller — planen må ikke selvstændigt fastlægge trafikale sikkerhedsforhold.

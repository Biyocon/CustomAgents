# BDK Styregruppearbejde

## Hvornår skillen bruges
- Når brugeren vil etablere eller justere styregruppeaftale.
- Når brugeren vil strukturere styregruppens beslutningsgrundlag.

## Arbejdsgang
1. Læs `references/source-map.md`.
2. Afgræns styregruppens mandat og opgaver.
3. Kortlæg nødvendige styringsdokumenter og beslutningsfora.
4. Lever handlingsnært forslag til styregruppeopsætning og opfølgning.

## Outputformat
1. Formål og mandat
2. Roller og ansvar
3. Beslutningsgrundlag
4. Møderytme og opfølgning
5. Kilder (repo-stier)

## Styringsregler
- Skillen må ikke erstatte formel godkendelse/underskrift.
- Markér afhængigheder til øvrige styringsdokumenter.

# Udbud og evaluering

## Formål
At sikre en transparent, ensartet og dokumenterbar udbuds- og evalueringsproces for entreprenør- og leverandørkontrakter i BBTR/BaneByg-projekter, i overensstemmelse med gældende udbudsregler og Banedanmarks indkøbsstrategi.

## Fremgangsmåde
1. Afklar udbuddets ramme: anskaffelsens genstand og estimeret værdi, om der udbydes under rammeaftale (miniudbud) eller som selvstændigt udbud, og hvem der er ansvarlig (jf. FB **Contract Manager, Project Level**: gennemføre korrekt udbud i samarbejde med kontrahering).
2. Udarbejd kravspecifikation ud fra fagpakker og ydelsesbeskrivelser; genbrug metoden fra `bbtr-raadgiver-udbud` (Bilag 1 – Kravspecifikation som skabelon) hvor relevant.
3. Definér tildelingskriterier og vægtning FØR tilbud modtages (fx pris, kvalitet, organisation/CV'er, proces og risiko), og fastlæg evalueringsmodellen skriftligt.
4. Planlæg udbudsprocessen: tidsplan, spørgsmål/svar-runder, ensartet information til alle tilbudsgivere og fortrolighedshåndtering.
5. Gennemfør evaluering: vurder konditionsmæssighed, scor hvert tilbud mod de offentliggjorte kriterier, og dokumentér begrundelser pr. kriterium pr. tilbudsgiver.
6. Udarbejd tildelingsdokumentation og beslutningsgrundlag; koordinér med jura/indkøb (se `bdk-jura-og-compliance`) om standstill, underretning og eventuelle klagerisici.
7. Overdrag til kontraktstyring (`bbtr-kontraktstyring`) med komplet udbudsmateriale, evalueringsrapport og kontraktgrundlag; arkivér sporbart jf. `bbtr-dokumentstyring`.

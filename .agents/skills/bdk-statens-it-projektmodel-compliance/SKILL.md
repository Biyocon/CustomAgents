# BDK Statens IT-projektmodel Compliance

## Hvornår skillen bruges
- Når brugeren vil afklare om et projekt er omfattet af Statens it-projektmodel.
- Når brugeren vil have overblik over It-rådskrav og rapporteringskrav.

## Arbejdsgang
1. Læs `references/source-map.md`.
2. Afgræns budget og anskaffelsesscope.
3. Kortlæg obligatoriske compliancekrav.
4. Lever kravmatrix og næste handlinger.

## Outputformat
1. Afgrænsning og tærskelvurdering
2. Obligatoriske krav
3. Rapporteringsforpligtelser
4. Næste handlinger
5. Kilder (repo-stier)

## Styringsregler
- Ved manglende økonomidata: returnér tydelige afklaringsspørgsmål.
- Skeln mellem "skal" og "kan".
- Præsenter ikke output som juridisk rådgivning.

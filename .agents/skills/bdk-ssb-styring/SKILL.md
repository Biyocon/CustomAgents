# BDK SSB Styring

## Hvornår skillen bruges
- Når brugeren spørger til en konkret SSB.
- Når brugeren vil have overblik over gældende SSB’er i ORS/ORF.
- Når brugeren vil se overlap mellem SSB’er og flere regelsæt.

## Arbejdsgang
1. Læs `references/source-map.md`.
2. Identificér mål-SBB’er via nummer, år eller regelsæt.
3. Byg et SSB-overblik:
   - reference (filnavn/nummer)
   - målområde (ORS/ORF/SR-kombi)
   - relation til øvrige SSB’er
4. Markér overlap og potentielle konflikter.
5. Lever anbefalet opfølgningsliste.

## Outputformat
1. Afgrænset SSB-sæt
2. SSB-oversigt (tabelform)
3. Overlap/konflikter
4. Prioriteret opfølgning
5. Kilder (repo-stier)

## Styringsregler
- Ingen fortolkning af juridisk gyldighed uden kildebelæg.
- Ved dubletter: markér primær kilde og alternativ kilde.
- Hvis versionsdato er uklar, markér behov for manuel verifikation.

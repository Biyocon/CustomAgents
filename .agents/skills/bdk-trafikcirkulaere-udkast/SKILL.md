# BDK Trafikcirkulære Udkast

## Hvornår skillen bruges
- Når brugeren vil udarbejde et TC-udkast.
- Når brugeren vil vælge korrekt TC-skabelon (ORS/ORF/SR-kombinationer).
- Når brugeren vil kvalitetssikre indholdsstruktur før godkendelse.

## Arbejdsgang
1. Læs `references/source-map.md`.
2. Afklar cirkulærekontekst:
   - regelsæt (ORS/ORF/SR-overlap)
   - geografisk/operativt scope
   - gyldighedsperiode og berørte enheder
3. Vælg korrekt skabelon.
4. Udarbejd udkast med obligatoriske sektioner fra vejledningen.
5. Kør en intern kvalitetstjekliste (klarhed, afgrænsning, konsekvens for drift).

## Outputformat
1. Valgt skabelon og begrundelse
2. TC-udkast (struktureret)
3. Manglende oplysninger, der kræver input
4. Kvalitetstjekresultat
5. Kilder (repo-stier)

## Styringsregler
- Skillen må ikke simulere formel godkendelse.
- Markér altid antagelser, hvis input mangler.
- Ved SR/ORS/ORF-overlap: vis tydeligt hvordan afgrænsning er håndteret.

# tools/python/

> Python-scripts til validering, beregning og automatisering.

## Indhold

- `validation_helpers.py` — generelle valideringsfunktioner
- `calculation_helpers.py` — matematiske hjælpere
- `excel_validator.py` — validering af Excel-filer mod schemas

## Brug

Alle scripts skal køres med `uv run`.

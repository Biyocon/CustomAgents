# CAD-eksportnoter

## Formål

Sikre korrekt eksport af tegninger fra AutoCAD/MicroStation til projektdokumentation.

## Anbefalede formater

- PDF (til review og arkivering)
- DWG/DGN (til videre bearbejdning)
- PNG/SVG (til web/præsentation)

## Tjekliste

- [ ] Korrekt målestok
- [ ] Layer-struktur er konsistent
- [ ] Tekster læselige
- [ ] Tegningshoved med projektinfo
- [ ] Revision og dato

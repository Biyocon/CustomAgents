# tools/

> Scripts, beregningshjælpere, validering og automatisering til Custom agent-harnesset.

## Formål

Denne mappe indeholder **aktive værktøjer**, som agenter kan bruge til at udføre beregninger, validere data, generere output og automatisere opgaver. Tools adskiller sig fra skills ved at være **eksekverbare**, ikke blot beskrivende.

## Undermapper

| Mappe | Indhold |
|-------|---------|
| `python/` | Python-scripts |
| `excel/` | Excel-hjælpere og -schemas |
| `excalidraw/` | Diagramskabeloner |
| `cad/` | CAD-eksportnoter |

## Hvad der ikke må gemmes her

- Projekt-specifikke datafiler
- Personfølsomme oplysninger
- Kontraktkritiske data
- Store binære filer

## Formål

Denne mappe indeholder standardiserede QA-tjeklister, der kan bruges til at kontrollere forskellige typer leverancer.

| Fil | Type | Status |
|-----|------|--------|

## Begrænsninger

- Tjeklister skal tilpasses projektspecifikke krav
- Dokumentér altid, hvilken tjekliste der blev brugt
- Marker usikkerhed i output

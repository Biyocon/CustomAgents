# Command: Hvad mangler vi?

> Identificér manglende data og afklaringer
> Trigger: "Hvad mangler vi?" eller tilsvarende

---

## Formål

Systematisk identificere, hvilke data der mangler for at gennemføre en opgave, ved at aktivere skill `data-gap-analyzer.md`.

## Trigger

- "Hvad mangler vi?"
- "Mangelliste"
- "Datamangel"
- "Afklaringer"

## Workflow

1. **Modtag opgavebeskrivelse**
   - Hvad skal laves?
   - Hvad er det nuværende datagrundlag?

2. **Aktivér skill**
   - `skills/data-gap-analyzer.md`

3. **Analyser mangler**
   - Identificér nødvendige data
   - Sammenlign med tilgængeligt
   - Kategorisér mangler
   - Formulér afklaringsspørgsmål
   - Foreslå antagelser

4. **Generér mangelliste**
   - Prioriteret liste
   - Afklaringsspørgsmål
   - Forslag til ansvarlige

## Output

- Datamangel-liste med kategorier
- Afklaringsspørgsmål
- Forslag til antagelser
- Konsekvenser af manglende data

## Begrænsninger

- Skel mellem manglende data og usikkerheder
- Marker altid, hvad der er antaget
- Opdater listen løbende

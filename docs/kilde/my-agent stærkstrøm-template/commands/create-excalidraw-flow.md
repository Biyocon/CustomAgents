# Command: Lav Excalidraw-flow

> Opret flowdiagramstruktur til Excalidraw
> Trigger: "Lav Excalidraw-flow" eller tilsvarende

---

## Formål

Generér en struktureret beskrivelse af et flowdiagram, der kan implementeres i Excalidraw, ved at aktivere skill `excalidraw-flow-generator.md`.

## Trigger

- "Lav Excalidraw-flow"
- "Lav flowdiagram"
- "Generér Excalidraw-struktur"

## Workflow

1. **Modtag input**
   - Systembeskrivelse
   - Komponenter og forbindelser

2. **Aktivér skill**
   - `skills/excalidraw-flow-generator.md`

3. **Generér struktur**
   - Definér flowtyper
   - Placér objekter
   - Definér forbindelser
   - Tilføj labels og noter

4. **Kontrollér**
   - Er alle komponenter inkluderet?
   - Er forbindelser korrekte?
   - Er labels konsistente?

## Output

- Excalidraw-kompatibel struktur
- Liste over objekter og forbindelser
- Forslag til lag og farver
- Diagramforklaring

## Begrænsninger

- Tekstbeskrivelse — ikke selve tegningen
- Bruger skal implementere i Excalidraw manuelt

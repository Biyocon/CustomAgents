# Plugin: Outlook

> Integration til Outlook
> Formål: Kommunikation og møder

---

## Beskrivelse

Outlook er en platform til e-mail og kalenderstyring. Plugin'et beskriver, hvordan agenten kan understøtte kommunikation via Outlook.

## Funktioner

- E-mail-kommunikation
- Mødeindkaldelser
- Kalenderstyring

## Agentens rolle

1. **Udkast til e-mails**
   - Formulér professionelle e-mails
   - Angiv vedhæftninger
   - Marker vigtighed

2. **Mødeindkaldelser**
   - Forslag til dagsorden
   - Identificér deltagere
   - Angiv formål

3. **Dokumentér**
   - Notér afsendte e-mails
   - Marker afklaringer

## Begrænsninger

- Agenten har ikke direkte adgang til Outlook
- Bruger skal sende e-mails manuelt
- Agenten kan hjælpe med udkast og struktur

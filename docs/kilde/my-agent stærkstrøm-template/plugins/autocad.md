# Plugin: AutoCAD

> Integration til AutoCAD
> Formål: Tegningsproduktion og -granskning

---

## Beskrivelse

AutoCAD er et CAD-værktøj til tegningsproduktion. Plugin'et beskriver, hvordan agenten kan understøtte arbejdet med AutoCAD.

## Funktioner

- Tegningsproduktion (SLD, tavleplaner, kabelplaner)
- Granskning og review
- Eksport/import af data

## Agentens rolle

1. **Forbered tegningsinput**
   - Generér koordinatlister
   - Beskriv komponentplacering
   - Angiv målestoksforhold

2. **Review tegninger**
   - Kontrollér konsistens med beregninger
   - Verificér symboler og betegnelser
   - Tjek målestoksforhold

3. **Dokumentér**
   - Notér tegningsnumre og -revisioner
   - Marker ændringer

## Begrænsninger

- Agenten har ikke direkte adgang til AutoCAD
- Bruger skal udføre tegning manuelt
- Agenten kan hjælpe med struktur og review

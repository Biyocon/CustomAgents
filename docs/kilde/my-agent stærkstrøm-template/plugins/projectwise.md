# Plugin: ProjectWise

> Integration til ProjectWise
> Formål: Projektstyring og dokumentkontrol

---

## Beskrivelse

ProjectWise er en platform til projektstyring og dokumentkontrol. Plugin'et beskriver, hvordan agenten kan understøtte arbejdet med ProjectWise.

## Funktioner

- Dokumentkontrol
- Versionsstyring
- Godkendelsesworkflows
- Adgangsstyring

## Agentens rolle

1. **Strukturér dokumenter**
   - Forslag til dokumentstruktur
   - Navngivningskonventioner
   - Godkendelsesflow

2. **Håndtér versioner**
   - Notér revisionshistorik
   - Marker godkendte versioner

3. **Dokumentér**
   - Notér dokumentlokationer
   - Angiv godkendelsesstatus

## Begrænsninger

- Agenten har ikke direkte adgang til ProjectWise
- Bruger skal uploade/hente dokumenter manuelt
- Agenten kan hjælpe med struktur og organisation

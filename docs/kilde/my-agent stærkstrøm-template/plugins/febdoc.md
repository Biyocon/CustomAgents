# Plugin: FEBDOC

> Integration til FEBDOC
> Formål: Load list-håndtering og beregninger

---

## Beskrivelse

FEBDOC er et værktøj til håndtering af belastningsdata og load lists. Plugin'et beskriver, hvordan agenten kan interagere med FEBDOC.

## Funktioner

- Import/eksport af load lists
- Beregning af strømme og effekter
- Validering af data

## Agentens rolle

1. **Forbered input**
   - Strukturér data i FEBDOC-kompatibelt format
   - Sørg for korrekte enheder og betegnelser

2. **Fortolk output**
   - Læs beregnede resultater
   - Identificér afvigelser

3. **Dokumentér**
   - Notér, hvilke beregninger der er udført i FEBDOC
   - Marker, hvad der kræver manuel verifikation

## Begrænsninger

- Agenten har ikke direkte adgang til FEBDOC
- Bruger skal udføre import/eksport manuelt
- Agenten kan hjælpe med at forberede og fortolke data

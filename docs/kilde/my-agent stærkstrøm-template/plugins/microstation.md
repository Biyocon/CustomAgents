# Plugin: MicroStation

> Integration til MicroStation
> Formål: Tegningsproduktion og -granskning

---

## Beskrivelse

MicroStation er et CAD-værktøj til tegningsproduktion, ofte brugt i infrastruktur- og anlægsprojekter. Plugin'et beskriver, hvordan agenten kan understøtte arbejdet med MicroStation.

## Funktioner

- Tegningsproduktion
- Granskning og review
- Eksport/import af data

## Agentens rolle

1. **Forbered tegningsinput**
   - Generér komponentlister
   - Beskriv lag og niveauer
   - Angiv målestoksforhold

2. **Review tegninger**
   - Kontrollér konsistens med beregninger
   - Verificér symboler og betegnelser

3. **Dokumentér**
   - Notér tegningsnumre og -revisioner
   - Marker ændringer

## Begrænsninger

- Agenten har ikke direkte adgang til MicroStation
- Bruger skal udføre tegning manuelt
- Agenten kan hjælpe med struktur og review

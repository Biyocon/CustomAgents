# Plugin: SharePoint

> Integration til SharePoint
> Formål: Dokumenthåndtering og deling

---

## Beskrivelse

SharePoint er en platform til dokumenthåndtering og samarbejde. Plugin'et beskriver, hvordan agenten kan understøtte arbejdet med SharePoint.

## Funktioner

- Dokumentopbevaring
- Versionsstyring
- Adgangsstyring
- Samarbejde

## Agentens rolle

1. **Strukturér dokumenter**
   - Forslag til mappestruktur
   - Navngivningskonventioner
   - Metadata

2. **Håndtér versioner**
   - Notér revisionshistorik
   - Marker seneste version

3. **Dokumentér**
   - Notér dokumentlokationer
   - Angiv adgangsrettigheder

## Begrænsninger

- Agenten har ikke direkte adgang til SharePoint
- Bruger skal uploade/hente dokumenter manuelt
- Agenten kan hjælpe med struktur og organisation

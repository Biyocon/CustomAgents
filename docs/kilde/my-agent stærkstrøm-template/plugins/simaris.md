# Plugin: Simaris

> Integration til Simaris
> Formål: Kabel- og beskyttelsesberegninger

---

## Beskrivelse

Simaris er Siemens' værktøj til kabeldimensionering, beskyttelsesberegning og selektivitetskontrol. Plugin'et beskriver, hvordan agenten kan interagere med Simaris.

## Funktioner

- Kabeldimensionering (tværsnit, spændingsfald)
- Beskyttelsesberegning (automatsikringer, relæer)
- Selektivitetskontrol
- Kortslutningsberegning

## Agentens rolle

1. **Forbered input**
   - Strukturér belastningsdata
   - Angiv installationsforhold
   - Specificér kabeltype og -materiale

2. **Fortolk output**
   - Læs anbefalede kabler
   - Verificér beskyttelsesindstillinger
   - Kontrollér selektivitet

3. **Dokumentér**
   - Notér Simaris-resultater
   - Marker afvigelser fra manuelle beregninger

## Begrænsninger

- Agenten har ikke direkte adgang til Simaris
- Bruger skal udføre beregninger manuelt
- Agenten kan hjælpe med input og fortolkning

# Skill: Work Description Writer

> Udarbejd teknisk arbejdsbeskrivelse
> Input: Opgave, omfang, forudsætninger, krav
> Output: Struktureret arbejdsbeskrivelse

---

## Formål

Generér en komplet teknisk arbejdsbeskrivelse med formål, omfang, forudsætninger, grænseflader, udførelseskrav, materialekrav, dokumentation, test og kvalitetssikring.

## Input

- Formål med arbejdet
- Omfang (hvad er inkluderet/ekskluderet)
- Forudsætninger
- Grænseflader til andre fag
- Udførelseskrav
- Materialekrav
- Dokumentationskrav
- Test- og idriftsættelseskrav
- Kvalitetssikringskrav

## Metode

1. **Formål**
   - Kort beskrivelse af arbejdets formål

2. **Omfang**
   - Hvad er inkluderet
   - Hvad er ekskluderet
   - Afgrænsning

3. **Forudsætninger**
   - Tekniske forudsætninger
   - Tidsmæssige forudsætninger
   - Organisatoriske forudsætninger

4. **Grænseflader**
   - Interface til andre fag
   - Interface til kunde
   - Interface til entreprenør

5. **Udførelseskrav**
   - Tekniske krav
   - Standarder
   - Sikkerhedskrav

6. **Materialekrav**
   - Materialer
   - Udstyr
   - Godkendelser

7. **Dokumentationskrav**
   - Tegninger
   - Beregninger
   - Certifikater

8. **Test og idriftsættelse**
   - Funktionsprøver
   - Dokumentation af test
   - Idriftsættelsesprocedure

9. **Kvalitetssikring**
   - Tjeklister
   - Review
   - Afleveringskrav

10. **Åbne punkter**
    - Afklaringsspørgsmål
    - Beslutninger

## Output

```markdown
# Arbejdsbeskrivelse: [Titel]

## 1. Formål
[Beskrivelse]

## 2. Omfang
### Inkluderet
- [Punkt 1]

### Ekskluderet
- [Punkt 1]

## 3. Forudsætninger
- [Forudsætning 1]

## 4. Grænseflader
| Fag | Ansvar | Bemærkning |
|---|---|---|

## 5. Udførelseskrav
- [Krav 1]

## 6. Materialekrav
- [Materiale 1]

## 7. Dokumentationskrav
- [Dokument 1]

## 8. Test og idriftsættelse
- [Test 1]

## 9. Kvalitetssikring
- [KS-punkt 1]

## 10. Åbne punkter
| ID | Punkt | Ansvarlig | Deadline |
|---|---|---|---|
```

## Begrænsninger

- Udkast til arbejdsbeskrivelse — skal reviewes
- Kræver projektspecifikke input
- Skal markeres som foreløbig

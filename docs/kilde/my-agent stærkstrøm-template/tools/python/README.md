# tools/python/

> Python-scripts til beregninger, validering og automatisering.

---

## Formål

Denne mappe indeholder Python-scripts, som agenten kan bruge til at udføre beregninger, validere data og automatisere opgaver.

## Indhold

| Fil | Formål | Status |
|-----|--------|--------|
| `load_list_tools.py` | Funktioner til load list-håndtering | Afventer |
| `excel_validator.py` | Validering af Excel-filer | Afventer |
| `calculation_helpers.py` | Hjælpefunktioner til beregninger | Afventer |

## Begrænsninger

- Scripts skal testes før brug i produktion
- Dokumentér altid, hvilket script der blev brugt
- Marker usikkerhed i output

# tools/excalidraw/templates/

> Genbrugelige diagramstrukturer til Excalidraw.

---

## Formål

Denne mappe indeholder skabeloner og strukturer til tekniske flowdiagrammer, der kan implementeres i Excalidraw.

## Indhold

| Fil | Type | Beskrivelse | Status |
|-----|------|-------------|--------|
| | | | Afventer |

## Standardstruktur

### Energiflow
- Forsyningstop (transformer, generator)
- Fordelingstavler
- Forbrugere

### Signaler
- SCADA/BMS
- Automatik
- Målere

### Styrestrømme
- Kontaktorer
- Relæer
- Sikringer

## Begrænsninger

- Skabeloner er vejledende
- Bruger skal implementere i Excalidraw manuelt
- Dokumentér altid, hvilken skabelon der blev brugt

# tools/cad/

> CAD-eksportnoter og importinstruktioner.

---

## Formål

Denne mappe indeholder noter og instruktioner til eksport og import af data mellem agenten og CAD-systemer (AutoCAD, MicroStation).

## Indhold

| Fil | Formål | Status |
|-----|--------|--------|
| `export-notes.md` | Noter til eksport af data til CAD | Afventer |

## Begrænsninger

- Agenten har ikke direkte adgang til CAD-systemer
- Bruger skal udføre import/eksport manuelt
- Dokumentér altid, hvilke data der er eksporteret/importeret

# MCP: Servers

> Liste over tilgængelige MCP-servere og deres konfiguration

---

## Formål

Dokumentere MCP-servere, som agenten kan forbinde til for at hente data, udføre beregninger eller interagere med eksterne systemer.

| Navn | Formål | Status | Konfiguration |

## Tilføjelse af nye servere

1. Beskriv serverens formål
2. Angiv forbindelsesdetaljer (URL, port, protokol)
3. Dokumentér tilgængelige tools og ressourcer
4. Angiv adgangsbegrænsninger

## Begrænsninger

- Ingen adgangsoplysninger (passwords, tokens) gemmes her
- Serverkonfiguration skal verificeres før brug
- Markér altid, hvis en server ikke er tilgængelig

## Tilføjelse af nye ressourcer

1. Beskriv ressourcens indhold
2. Angiv format (JSON, CSV, XML, etc.)
3. Dokumentér opdateringsfrekvens
4. Angiv adgangsbegrænsninger

## Begrænsninger

- Fortrolige data skal markeres tydeligt
- Ressourcer skal verificeres før brug
- Angiv altid datakilden

# Claude Memory

Append-only project memory.

This file is meant to be updated automatically by a git `post-commit` hook.
Install it with:

- `{{INSTALL_HOOK_CMD}}`   (e.g. `npm run claude:install-hook`)

Each automatic entry records:

- timestamp
- commit hash
- commit message

> Set up the hook so every commit appends `YYYY-MM-DD HH:MM | <hash> | <subject>`
> here. If you have no remote, history is local-only — say so explicitly.

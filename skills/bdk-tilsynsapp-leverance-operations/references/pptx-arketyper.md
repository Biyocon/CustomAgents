# PPTX-arketyper v2.0 - Tilsynsappen

Baseret paa maskinanalyse af 32 PPTX-filer (805 slides).

## 1. Traening / Modulundervisning
- **Filer:** Dag 1-3 moduler, Modul 3-5, Oevelser til traening
- **Slides:** 258 i 5 moduler (stoerste kategori)
- **Moenster:** Laeringsmaal, demo-flow med screenshots, oevelser, evalueringspunkt
- **Font:** Segoe UI + Ink Free (bevidst haandskrifts-element)
- **Visuel taethed:** Hoej (~3-5 billeder/slide i screencast-sektioner)
- **Guldstandard:** Modul 4 - App og dashboard.pptx (74 slides, 199 billeder)
- **Noter-daekning:** Lav (22/74 i Modul 4)

## 2. Onboarding / Superbruger
- **Filer:** Onboarding af nye superbrugere i Tilsynsappen.pptx
- **Slides:** 36 (ekskl. duplikat)
- **Moenster:** Rolleintro, trin-for-trin, 7 diagrammer, supportkanaler
- **Font:** Segoe UI (72), Arial (12 = importstoj)
- **Visuel taethed:** Middel (diagrammer > screenshots)
- **Guldstandard:** Onboarding af nye superbrugere i Tilsynsappen.pptx

## 3. Sikkerhedsdag / Governance
- **Filer:** Ro-Rg Sikkerhedsdag (2024 + 2025), Sikkerhedsdag Arbejdsmiljo
- **Slides:** 134 (48 + 67 + 19)
- **Moenster:** Risiko, ansvar, haendelser, beslutninger, laering
- **Font:** Stoerst diversitet (5+ fonte: Segoe UI, Arial, Times, Aptos, Avenir)
- **Visuel taethed:** Middel-hoej (diagrammer, fotos fra mark)
- **OBS:** Primaer normaliseringskandidat pga. fontdiversitet

## 4. Tavleskabeloner
- **Filer:** Skabeloner_byggechef, Skabeloner_fagbyggeleder (x4), Skabeloner_sikkerhed, Tomme Skabeloner
- **Slides:** ~86 (6+12+24+8+23+8+7 = ~88 ekskl. overlap)
- **Moenster:** Tabelbaserede — stadie-registrering, fravaer, aktionslister
- **Font:** Segoe UI + Ink Free (haandskrift i udfyldningsfelter)
- **Visuel taethed:** Lav (tabeller, ingen/faa billeder)
- **Varianter:** Per fagomraade: Jord/Afvanding, Sikring, Spor + Byggechef + Sikkerhed

## 5. Bestilling / Logistik
- **Filer:** Bestilling-til-1-projekt, Endelig-bestilling, Bestillingsoverblik, Magneter til produktion, Omkostningssoverblik (v1+v2)
- **Slides:** ~38
- **Moenster:** Bestillingslister, omkostningsopgoerelser, fraktprocesser
- **Font:** Segoe UI (primaer), Arial (sekundaer)
- **Visuel taethed:** Lav (tabeller og lister)
- **OBS:** Omkostningssoverblik har 2 charts (eneste filer med charts)

## 6. Overlevering / Sidemandsoplaring
- **Filer:** Erfaringer med Tilsynsapp, Sidemandsoplaring m. OpEx
- **Slides:** 15 (3 + 12)
- **Moenster:** Erfaringsopsamling, overdragelse fra OpEx til S&U, dato-specifik
- **Font:** Segoe UI (primaer)
- **Visuel taethed:** Middel

## 7. Governance-roller / Introduktion
- **Filer:** IPM_Ulla, Intro til rollen som lev+digi
- **Slides:** 14 (8 + 6)
- **Moenster:** Rollebeskrivelse, organisationsdiagram, ansvarsfordeling
- **Font:** Segoe UI + Arial
- **Visuel taethed:** Middel

## 8. Roadmap / Videreudvikling
- **Filer:** Tilsynsapp scope, Tilsynsapp videreudviklings forslag LDK
- **Slides:** 71 (10 + 61)
- **Moenster:** Gap-analyse, 3-fase roadmap, aendringsforslag, prioritering
- **Font:** Tilsynsapp scope: Arial (152!) > Segoe UI (72) — font-anomali
- **Font:** LDK: Arial (249!) > Segoe UI (218) — ogsaa font-anomali
- **OBS:** Begge roadmap-filer har Arial som PRIMAER font — klar normaliseringskandidat

## 9. App-procesflow

- **Slides:** 11

- **Guldstandard:** Tilsynsapp og tavle process.pptx (BKP-kobling via Brutto KP)

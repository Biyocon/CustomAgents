Byg/redesign et “Shortcut-style Spreadsheet Workspace” modul til OmniAI.

Målet er at implementere et moderne Excel-lignende workspace med AI-assistent, filhåndtering, shortcuts, skills, tutorials og chat history.

Brug eksisterende OmniAI shell, routing, tokens og komponentstil. Designet skal matche de vedhæftede screenshots: rent, professionelt, spreadsheet-first, med diskret AI-panel og store modal flows.

Implementér følgende centrale områder:

1. Spreadsheet Editor Shell
2. Right-side AI Workspace Panel
3. My Shortcuts Modal
4. Open File Modal
5. New File Modal
6. Ribbon Tabs og Toolbar States
7. File History, Skills, Tutorials og Prompt Library

========================================
1. OVERORDNET PRODUKTKONCEPT
========================================

Byg et webbaseret spreadsheet workspace inspireret af Excel/Shortcut.

Brugeren skal kunne:
- Åbne eksisterende regneark
- Oprette nye regneark
- Eksportere filer
- Navigere i et Excel-lignende grid
- Skifte ribbon tabs
- Bruge AI-assistent i højre sidepanel
- Tilgå shortcuts, prompt library, skills, tutorials og chat history
- Arbejde med spreadsheet-filer i et workflow, der kombinerer klassisk spreadsheet UI med AI-agent handlinger

Primær følelse:
- Excel-lignende funktionalitet
- Minimalistisk web-app layout
- AI-panel som produktivitetslag ovenpå regnearket
- Hurtig fil- og template-orienteret workflow

18. DATA MODEL
========================================

Brug Zustand eller eksisterende state management.

Workbook schema:

{
  id: string
  name: string
  sheets: Sheet[]
  activeSheetId: string
  lastOpened: string
  sizeKb: number
  createdAt: string
  updatedAt: string
}

Sheet schema:

{
  id: string
  name: string
  cells: Record<string, CellData>
  rowCount: number
  columnCount: number
}

CellData schema:

{
  value: string
  formula?: string
  format?: CellFormat
}

CellFormat schema:

{
  bold?: boolean
  italic?: boolean
  underline?: boolean
  fontFamily?: string
  fontSize?: number
  textColor?: string
  fillColor?: string
  align?: "left" | "center" | "right"
  verticalAlign?: "top" | "middle" | "bottom"
  numberFormat?: "general" | "currency" | "percent" | "date" | "number"
}

Spreadsheet UI state:

{
  activeWorkbookId: string
  activeSheetId: string
  activeCell: string
  selectedRange?: string
  formulaBarValue: string
  zoom: number
  activeRibbonTab: "Home" | "Insert" | "Page Layout" | "Formulas" | "Data" | "View" | "Settings"
  isWorkspaceOpen: boolean
  workspaceMode: "assistant" | "workspace"
  activeWorkspaceTab: "Files" | "Skills" | "Templates" | "Memories" | "Backups"
  isOpenFileModalOpen: boolean
  isNewFileModalOpen: boolean
  isShortcutsModalOpen: boolean
}

ShortcutPrompt schema:

{
  id: string
  title?: string
  prompt: string
  createdAt: string
  updatedAt: string
}

Skill schema:

{
  id: string
  title: string
  description: string
  tags: string[]
  scope: "personal" | "default"
  status: "active" | "draft"
}

Tutorial schema:

{
  id: string
  title: string
  imageUrl?: string
  tags: string[]
  category: "Official" | "Personal"
}

ChatHistory schema:

{
  id: string
  workbookName: string
  title: string
  createdAt: string
  relativeTime: string
}

Implementeringen er godkendt når:

- Spreadsheet editor visuelt matcher screenshots.

- Home, Page Layout, Formulas, View og Settings tabs har realistiske command controls.
- Grid viser kolonner, rækker, celler, headers, aktiv celle og scrollbars.
- Formula bar og name box er synlige og funktionelle.

- Prompt Library kan oprette prompt.
- Tutorials tab viser official tutorial cards.
- Chat History tab viser søgbar, grupperet historik.
- Open File modal viser søgbar filhistorik.
- New File modal viser tre store action cards.

Byg/redesign et “Shortcut-style Spreadsheet Workspace” modul til OmniAI.

Målet er at implementere et moderne Excel-lignende spreadsheet workspace med AI-assistent, filhåndtering, shortcut-bibliotek, skills, tutorials og chat history.

Brug eksisterende OmniAI shell, routing, tokens, ikoner og komponentstil. Modulet skal visuelt matche de vedhæftede screenshots: spreadsheet-first, kompakt, professionelt, med klassisk Excel-ribbon og et diskret AI-lag.

Byg ikke rigtig Excel-kompatibilitet endnu. Byg en funktionel frontend-prototype med realistisk mock data, tydelig state model og UI-interaktioner, så backend/API kan kobles på senere.

Routes:
- /spreadsheet
- /spreadsheet/:fileId
- /spreadsheet/new
- /spreadsheet/open
- /spreadsheet/templates
- /spreadsheet/shortcuts

Hvis eksisterende routing-konventioner bruges i OmniAI, følg dem, men funktionaliteten skal svare til ovenstående.

12. Functional Acceptance Criteria
========================================

Godkend kun implementeringen når:

- /spreadsheet viser spreadsheet editoren.
- Top file bar matcher screenshots.
- Open File, New File og Export File findes.
- Ribbon tabs kan skiftes.
- Home, Page Layout, Formulas, View og Settings viser realistiske controls.
- Grid viser kolonner A–AE og mindst 52 rækker.
- Aktiv celle kan vælges.
- Name box opdateres ved cellevalg.
- Formula bar viser og ændrer celleværdi.
- Arrow keys, Tab, Enter og Escape virker basalt.
- Sheet tab “Sheet1” findes.
- Zoom slider ændrer zoom state.
- Workspace panel kan vises/skjules.
- Assistant mode og dark Workspace mode findes.
- My Shortcuts modal har alle fire tabs.
- Prompt Library kan oprette en prompt.
- Skills tab viser 10 mock skills.
- Tutorials tab viser tutorial cards i scrollbart grid.
- Chat History tab har søgning og grupperede conversations.
- Open File modal viser søgbar filhistorik.
- New File modal viser tre store cards.
- Alle modals kan lukkes med X og Escape.
- Layout er responsive.
- Mock data er centraliseret.
- Ingen spredt hardcoding i UI-komponenterne.
- Ingen backend/API antages; alt kører frontend-only med mock state.
